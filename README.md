# Elite AI

Elite AI is a full-stack web application for AI training job opportunities.

Users can register, log in, complete a profile, upload a CV, pay for CV verification, access gigs after approval, and later manage payout methods.

## Tech Stack

### Frontend
- React
- JavaScript

### Backend
- Flask
- Pipenv
- PostgreSQL
- Marshmallow

## Development Environment

This project is being built in WSL Ubuntu using VS Code.
